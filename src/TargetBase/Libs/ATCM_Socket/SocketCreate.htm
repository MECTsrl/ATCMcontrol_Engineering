SocketCreate

La funzione SocketCreate permette di creare un socket.

 

Sintassi

 

 

ret_code := SocketCreate (AddressFamily, ProtocolType, Socket) ;

 

 

Parametri di ingresso

 

AddressFamily (DINT)    Famiglia di indirizzi

ProtocolType (DINT)       Tipo di protocollo

 

Parametri di uscita

 

Socket (UDINT)                  Socket creato dalla funzione

 

 

Uso

 

La funzione SocketCreate permette di creare un socket, che può essere utilizzato per inviare/ricevere informazioni su una rete utilizzzando il protocollo TCP oppure il protocollo UDP.

 

Il parametro AddressFamily  può assumere i seguenti valori:

 

AddressFamily

Valore

Descrizione

SCK_AF_IPv4

0

Internet Protocol versione 4 (IPv4)

 

Il parametro ProtocolType può assumere i seguenti valori:

 

ProtocolType

Valore

Descrizione

SCK_PROTO_TCP

1

Transmission Control Protocol (TCP)

SCK_PROTO_UDP

2

User Datagram Protocol (UDP)

 

Un socket per default viene creato in modalità bloccante.

Se si desidera utilizzare il socket in modalità non bloccante allora, dopo aver creato il socket, occorre chiamare la primitiva SocketEnableNonblocking.

 

 

Valori di ritorno

Valore (HEX)

Descrizione

0

Funzione eseguita correttamente

0x00390001

Parametro AddressFamily non valido

0x00390002

Parametro ProtocolType non valido

0x0039001E

La rete non sta funzionando

  0x00390003

Troppi socket contemporaneamente in uso

  0x00390008

Buffer space esaurito

0x00390009

Questo tipo di socket non è supportato

0x00390020

Errore generico; questo errore non dovrebbe mai verificarsi

 

 

     

Esempio

 

VAR

Ret_code:  DWORD ;

Socket : UDINT ;

 

END_VAR

 

Ret_code := SocketCreate(SCK_AF_IPv4, SCK_PROTO_TCP, Socket);
