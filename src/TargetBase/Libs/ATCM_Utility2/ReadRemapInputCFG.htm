ReadRemapInputCFG

La funzione ReadRemapInputCFG permette di caricare in memoria un file di definizioni di remap di input digitali.

 

 

Sintassi

 

 

ret_code := ReadRemapInputCFG (Filename) ;

 

 

Parametri di ingresso

 

            Filename (STRING)        nome, completo di path, del file di definizioni di remap di input

                                                digitali

 

Modalità di esecuzione

 

Wait

 

 

Uso

 

La funzione ReadRemapInputCFG permette di caricare in memoria un file di definizioni di remap di input digitali. Il file di definizioni di remap di input digitali, completo di path, va passato alla funzione nel parametro Filename

 

Questa funzione consente di rimappare automaticamente input digitali fisici (variabili IW) su input digitali logici (variabili MW).

Il file di definizione del remap degli input digitali deve essere un file XML come questo:

 

<?xml version="1.0" encoding="UTF-8" ?>

<InputRemapConfig FileVersion="1" MaxLineNumber="128">

   <InRemap PhysWord="99" PhysBit="0" LogicWord="23" LogicBit="15" Mode="Copy"/>

   <InRemap PhysWord="99" PhysBit="1" LogicWord="23" LogicBit="14" Mode="Ignore"/>

   <InRemap PhysWord="99" PhysBit="2" LogicWord="23" LogicBit="13" Mode="SetToTrue"/>

   <InRemap PhysWord="99" PhysBit="3" LogicWord="23" LogicBit="12" Mode="SetToFalse"/>

   <InRemap PhysWord="99" PhysBit="4" LogicWord="23" LogicBit="11" Mode="Negate"/>

</InputRemapConfig>

 

Per esempio la riga

   <InRemap PhysWord="99" PhysBit="0" LogicWord="23" LogicBit="15" Mode="Copy"/>

consente di copiare automaticamente un bit di input fisico (il bit 0 della word IW99) su un bit di input logico (il bit 15 della word MW23).

 

In realtà esistono diversi modi di esecuzione (Mode):

 

Mode

Descrizione

Copy

Al bit LogicBit della word LogicWord viene assegnato il valore del bit PhysBit della word di input PhysWord

Ignore

Non viene fatto nulla

SetToTrue

Al bit LogicBit della word LogicWord viene assegnato il valore true, a prescindere dal valore del bit PhysBit della word di input PhysWord

SetToFalse

Al bit LogicBit della word LogicWord viene assegnato il valore false, a prescindere dal valore del bit PhysBit della word di input PhysWord

Negate

Al bit LogicBit della word LogicWord viene assegnato il valore negato del bit PhysBit della word di input PhysWord

 

 

Si noti che la funzione ReadRemapInputCFG carica solo in memoria il file di definizione del remap degli input digitali, ma non esegue veramente il remap.

Per eseguire il remap degli input digitali occorre chiamare esplicitamente la funzione RemapInput().

 

Il numero di input digitali da rimappare non deve superare il valore del campo MaxLineNumber, definito nelle prime righe del file XML.

 

Nel caso in cui la funzione ReadRemapInputCFG venga chiamata più di una volta le ultime definizioni NON vengono “aggiunte” a quelle relative alle chiamate precedenti in modo incrementale, bensì a ogni chiamata della funzione innanzitutto vengono cancellate tutte le definizioni di remap eventualmente esistenti e poi vengono caricate in memoria solo le definizioni presenti nel file passato come parametro nella chiamata della funzione.

 

Valori di ritorno

Nella successiva tabella sono riportati i possibili valori assunti dalla variabile ret_code.

ret_code (HEX)

 

0x00000000

Funzione eseguita senza errori

0x0015006A

Errore di allocazione di memoria

0x001500AA

Il nome del file specificato nel parametro Filename è troppo lungo

0x001500AB

Il file specificato nel parametro Filename non esiste

0x001500AC

Errore nel caricamento del file specificato nel parametro Filename

0x001500AD

Il sistema non è in grado di leggere il file perché la versione del file (definita nell’attributo FileVersion presente all’interno del file) è troppo avanzata

0x001500AE

Errore sintattico all’interno del file XML

0x001500AF

All’interno del file esistono dei campi con valori al di fuori dal range consentito

0x001500B0

Il numero di righe del file (numero dei bit da rimappare) è superiore al valore del campo MaxLineNumber, definito nelle prime righe del file XML.

 

 

Esempio

Ret : dword;

Ret := ReadRemapInputCFG (‘\SSD\osai\InputRemapFile.xml’);
