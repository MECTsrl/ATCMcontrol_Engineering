XMLRemove

La funzione XMLRemove rimuove un elemento da un file XML

 

 

Sintassi

 

 

ret_code := XMLRemove (XMLhandle, XmlPath) ;

 

 

 

Parametri di ingresso

 

            XMLhandle(DINT)           identificatore univoco del file XML.

            XmlPath (STRING)         path XML dell’elemento da rimuovere

 

Parametri di uscita

 

 

Modalità di esecuzione

 

Wait

 

 

Uso

 

Tale funzione permette di rimuovere un elemento da un file XML.

Il file XML su cui operare è identificato dal parametro XMLhandle: tale parametro viene ritornato dalla funzione XMLOpen alla apertura del file.

Il parametro XmlPath consente di selezionare l’elemento da rimuovere all’interno del file XML.

Se al di sotto dell’elemento selezionato esiste un sotto-albero di elementi anche tutto il sotto-albero viene rimosso.

 

 

Valori di ritorno

 

Nella successiva tabella sono riportati i possibili valori assunti dalla variabile ret_code

           

ret_code (HEX)

Descrizione

0x00000000

Funzione eseguita senza errori

0x00380002

Il parametro XMLhandle non è valido

0x0038000A

L’elemento identificato da XmlPath non esiste nel file XML

 

 

 

 

 

 

 

 

 

Esempio

 

Ret       : dword;

XMLhandle : dint;

…

(* apre il file SSD\osai\file.xml in lettura/scrittura, rimuove l’elemento elem2, salva il file e infine lo chiude *)

 

Ret := XMLOpen(‘\SSD\osai\file.xml’, XML_FMODE_READ_WRITE, XMLhandle);

Ret := XMLRemove(XMLhandle, '/top/elem2');

Ret := XMLSave(XMLhandle);

Ret := XMLClose(XMLhandle);

…

Questo è il contenuto del file SSD\osai\file.xml prima della esecuzione della XMLRemove:

<?xml version="1.0"?>

<top>

  <elem1>10</elem1>

  <elem2>11</elem2>

</top>

 

Questo è il contenuto del file SSD\osai\file.xml dopo la esecuzione della XMLRemove:

<?xml version="1.0"?>

<top>

  <elem1>10</elem1>

</top>
